# write-verified-advisory

Produce professional, evidence-based technical advisories and research memos. This skill enforces a dual-output structure: a polished stakeholder report and a raw verification audit for the author.

Every technical claim must be backed by a live, verified source — no citing from training data. The skill mandates a verification hierarchy from browser automation down to raw curl inspection, and prohibits claims based solely on search summaries.

Cross-platform — works with any agent that supports the SKILL.md format.

## Install

```bash
pipx install echomodel
em skills install write-verified-advisory
```

See the [echoskill README](../../README.md#quick-start) for full setup.
