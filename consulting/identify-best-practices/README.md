# identify-best-practices

Identify and compare your approach against industry best practices. The agent understands your perspective, then researches how leaders in the relevant problem space think — what patterns and tools they use, what language describes the problem, what's considered an antipattern, and what you may not be accounting for.

Not sycophantic, not dismissive. The agent won't validate your approach just because you proposed it, but it also won't demand you copycat established practice when your context genuinely calls for something different. It helps you distinguish between reinventing the wheel and solving a problem the wheel wasn't designed for.

Includes a structured documentation workflow: the exploration produces a concise decision record (decided approach, alternatives considered, what was confirmed/challenged, go-forward plan) with guidance on where each piece of documentation belongs based on audience and repo visibility.

Cross-platform — works with any agent that supports the SKILL.md format.

## Install

```bash
pipx install echomodel
em skills install identify-best-practices
```
