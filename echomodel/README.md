# echomodel

Skills for building and shipping MCP-powered solutions using the [echomodel](https://echomodel.ai) ecosystem. These skills guide agents through creating, structuring, and deploying Python MCP servers — from first file to production.

## Skills

| Skill | Description |
|-------|-------------|
| [author-mcp-app](author-mcp-app/SKILL.md) | Build, structure, migrate, or review Python MCP servers and web APIs |

## Ecosystem

These skills promote the [mcp-app](https://github.com/echomodel/mcp-app) config-driven framework for fast time-to-market, while remaining compatible with standard [FastMCP](https://github.com/modelcontextprotocol/python-sdk) for teams that prefer direct control.

Key ecosystem components:
- **[mcp-app](https://github.com/echomodel/mcp-app)** — Config-driven MCP server framework (YAML config, auto tool discovery, built-in auth)
- **[gapp](https://github.com/echomodel/gapp)** — Rapid deployment to serverless Cloud Run, designed with MCP servers in mind
- **[app-user](https://github.com/echomodel/app-user)** — JWT auth and per-user data storage
