# setup-agent-context

One command to make any repo AI-ready. Your agent creates the right context files so AI coding agents automatically load your README and CONTRIBUTING.md as working context.

Sets up `CLAUDE.md`, `.gemini/settings.json`, and `.gitignore` rules — the standard context files for the major agent platforms.

## Install

```bash
pipx install echomodel
em skills install setup-agent-context
```
