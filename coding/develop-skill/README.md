# develop-skill

Write an agent skill once, run it on any AI agent that supports the SKILL.md format. This skill teaches your agent how to build portable skills without platform-specific conditionals.

Covers the full lifecycle: SKILL.md structure, frontmatter conventions, what's portable vs platform-specific, and how to handle the differences without forking.

## Install

```bash
pipx install echomodel
em skills install develop-skill
```
